Programming Fundamentals in Python

1. Introduction

Programming is the process of giving step-by-step instructions to a computer to perform a task.

Python is one of the most popular programming languages today, known for its simplicity, readability, and versatility.

Why Python?

  • High-level: You can think in human terms, not in binary or memory registers.
  • Interpreted: No compilation step needed; Python executes code line by line.
  • Cross-platform: Works on Windows, macOS, Linux, even in mobile/web apps.
  • Versatile: Useful in automation, data science, artificial intelligence, web development, and more.

Example:

print("Welcome to Python Programming")

Output:

Welcome to Python Programming

2. Analogies and Examples

Learning programming can be simplified through real-world analogies.

Analogy (Recipe):

  • Recipe steps → Program instructions
  • Ingredients → Input data
  • Final dish → Program output

Example (Tea Recipe → Python Code):

def make_tea(water, tea_leaves, milk, sugar):
    return f"Tea ready with {water}ml water, {tea_leaves} spoons of tea, {milk}ml milk, and {sugar} spoons of sugar."
print(make_tea(200, 2, 50, 1))

Output:

Tea ready with 200ml water, 2 spoons of tea, 50ml milk, and 1 spoons of sugar.

Just like customizing a recipe, in programming, you pass inputs (parameters) to functions to change the result.

3. Compilation and Runtime Terms

Compilation

Some languages (C, Java) require code to be compiled (converted into machine code) before running. Compilation checks for all errors first and then creates a machine code executable.

Runtime

Runtime is when the program is actually running on the computer. Errors at runtime are called runtime errors (e.g., dividing by zero).

Python’s Execution Model

Python uses an interpreter instead of a compiler. The interpreter executes code line by line, making it easier to debug but slower than compiled languages.

Example

print("Hello World")

At runtime, Python executes this line and prints the message.

4. Functions

A function is a reusable block of code that performs a specific task.

Key Points:

  • Helps avoid repetition.
  • Makes programs modular (divided into small parts).
  • Accepts input (parameters) and may return output (result).

Analogy:

A vending machine:

  • Input = Coins (parameters)
  • Processing = Select drink/snack
  • Output = Snack (return value)

Syntax:

def function_name(parameters):
    # code block
    return result

Example:

def greet(name):
    return "Hello, " + name

print(greet("Saikumar"))

Output:

Hello, Saikumar

5. Input Parameters

Parameters are values you pass into a function so it can use them in its operation.

Analogy:

A blender needs fruits, milk, and sugar as inputs. Without them, it can’t make juice.

Syntax:

def function_name(param1, param2):
    # use the parameters
    return something

Example:

def add_numbers(a, b):
    return a + b

print(add_numbers(5, 10))

Output:

15

6. Return Statement

The return statement is used to send a value back from a function to the caller.

Key Notes:

  • Once return executes, the function ends immediately.
  • A function without return returns None by default.

Syntax:

def function_name():
    return value

Example:

def square(n):
    return n * n

result = square(4)
print(result)

Output:

16

7. Print Statement

The print() function is used to display output on the screen.

Difference Between print() and return:

  • print() → Shows result on screen.
  • return → Sends result back to be stored/used later.

Syntax:

print(value)

Example:

def example():
    print("This just shows text")
    return "This sends value back"

x = example()
print("Returned value:", x)

Output:

This just shows text
Returned value: This sends value back

Full Example Combining All Concepts

A complete program combining function, parameters, return, and print.

# Define function with parameters, return, and print
def calculate_area(length, width):
    area = length * width
    return area   # returning result

# Inputs
length = 5
width = 3

# Call function
result = calculate_area(length, width)

# Print output
print("The area of rectangle is:", result)

Output:

The area of rectangle is: 15

Key Takeaways

  • Python is interpreted, simple, versatile.
  • Functions help write reusable, modular code.
  • Parameters customize function behavior.
  • return sends back results; print() displays them.
  • Programs are like recipes – steps, inputs, and final output.
